# Contributing to BGMI Name Stylist

First off, thank you for considering contributing to BGMI Name Stylist! It's people like you that make BGMI Name Stylist such a great tool for the community.

## How Can I Contribute?

### Reporting Bugs
This section guides you through submitting a bug report for BGMI Name Stylist.
* **Use a clear and descriptive title** for the issue to identify the problem.
* **Describe the exact steps which reproduce the problem** in as many details as possible.
* **Provide specific examples to demonstrate the steps**.

### Suggesting Enhancements
* **Use a clear and descriptive title** for the issue to identify the suggestion.
* **Provide a step-by-step description of the suggested enhancement** in as many details as possible.
* **Explain why this enhancement would be useful** to most BGMI Name Stylist users.

### Pull Requests
* Fill in the required template.
* Do not include issue numbers in the PR title.
* Follow the styleguides.
* After you submit your pull request, verify that all status checks are passing.

## Development Setup

1. Fork the repo and create your branch from `main`.
2. Install dependencies in the root directory: `npm install`.
3. Start the dev server: `npm run dev`.
4. Make your changes and test them thoroughly.
5. Commit your changes and open a Pull Request.

## Styleguides
### Git Commit Messages
* Use the present tense ("Add feature" not "Added feature")
* Use the imperative mood ("Move cursor to..." not "Moves cursor to...")
* Limit the first line to 72 characters or less
* Reference issues and pull requests liberally after the first line

Thank you for your contribution!
