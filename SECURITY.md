# Security Policy

## Supported Versions

Only the latest main branch version is currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| Main    | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

If you discover any security related issues, please do not use the issue tracker. Instead, please email the project maintainer privately. We take security vulnerabilities very seriously and will work to resolve the issue as quickly as possible.
